Architecture
Swift / SwiftUI

Mac App

  • NSPanel — floating popup window
  • NSStatusItem — menu bar icon
  • CGEventTap — global hotkey
  • FSEventStream — watch ~/
  • NSTask — manage server process
  • URLSession — REST calls
Python / FastAPI

Companion Server

  • GET /api/search
  • GET /api/thumbnail
  • POST /api/reveal
  • GET /api/scan
  • Port 58421 (fixed)
  • Auto-started by app
Shared / On-disk

llmind-cli

  • XMP embeddings in files
  • Hybrid search logic
  • keyword_score()
  • cosine_similarity()
  • Ollama (optional)
  • .llmind.* file format
